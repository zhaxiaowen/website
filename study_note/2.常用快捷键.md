# 常用快捷键

#### vim

```
D  #删除到行位
x  #删除光标后一个字符,3x,删除光标后面3个字符
X  #删除光标前一个字符
d0 #删除光标至行首
d$ #删除光标至行尾
daw #删除一个单词
dw #删除光标后的一个单词
de #删除光标前的一个单词
0  #光标移至行首
$  #光标移至行位
u	 #撤销一次操作

#批量添加注释
ctrl + v  #进入块选择模式,选择好要操作的行,
I   #大写I进入行首插入模式,插入#,
按两下ESC
```

#### linux终端

```
ctrl + 左右键  #在单词之间跳转,mac是 option键
ctrl + a     #跳到行首
ctrl + e		 #跳到行尾
ctrl + u     #删除当前光标前面的文字(剪切)
ctrl + k     #删除当前光标后面的文字(剪切)
ctrl + y     #粘贴

ctrl + w     #删除光标前的一个单词
```
